# Анализ тональности текстовых отзывов (Olist / Brazilian E-Commerce)

В данном проекте решается задача **анализа тональности текстовых отзывов** на основе набора данных **Brazilian E-Commerce Public Dataset by Olist** (раздел *Order Reviews Dataset*).  
Цель — **определить эмоциональную окраску отзыва по его тексту** с применением методов **NLP** и **машинного обучения**.

---

##  Описание
В ноутбуке реализован полный пайплайн:
- загрузка и первичный анализ данных отзывов;
- **предобработка текста** (определение языка, очистка, нормализация, токенизация);
- формирование **целевой переменной** на основе пользовательских оценок;
- обучение **модели классификации** тональности;
- **оценка качества** и анализ результатов.

---

##  Данные
Источник: **Brazilian E-Commerce Public Dataset by Olist**  
Используется таблица/часть датасета: **Order Reviews Dataset** (отзывы покупателей).
### Ссылка для скачивания датасета
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce?select=olist_order_reviews_dataset.csv

---

##  Постановка задачи
**Вход:** текст отзыва 
**Выход:** класс тональности (негативная / нейтральная / позитивная)

---

##  Предобработка (примерный список)
- удаление пустых/дублирующихся записей;
- очистка текста (пунктуация, спецсимволы, лишние пробелы);
- приведение к нижнему регистру;
- удаление стоп-слов;
- векторизация текста (FastText, BERT).

---

##  Модель
Обучается модель **классификации** для предсказания тональности.  
В качестве базовой реализации часто используется связка:
- векторизация FastText, BERT
- классификатор (Logistic Regression / CatBoost)


---

##  Оценка качества
Для оценки качества применяются стандартные метрики классификации:
- Accuracy
- Precision / Recall / F1-score
- Confusion Matrix
