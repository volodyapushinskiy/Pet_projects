# Классификация писем: спам / не спам

Проект посвящён задаче **бинарной классификации** электронных писем на категории **Spam** и **Non-spam**.  
Цель — построить модель машинного обучения, которая по тексту письма автоматически определяет, является ли оно спамом.
---

##  Описание
В работе реализован полный ML/NLP пайплайн:
- загрузка и первичный анализ датасета с письмами;
- **предобработка текста** (очистка, нормализация, удаление шумов);
- преобразование текста в числовые признаки (**Bag of Words**);
- обучение моделей классификации;
- оценка качества и анализ ошибок.


---

##  Постановка задачи
**Вход:** текст письма
**Выход:** метка класса:
- `1` — Spam  
- `0` — Non-spam
