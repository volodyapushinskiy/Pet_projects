# Задача регрессии: прогноз цены ноутбука

В данном проекте решается задача **регрессии** — предсказание **стоимости ноутбука** на основе его технических характеристик.  
Цель работы — построить модель машинного обучения, которая по набору признаков оценивает **цену устройства**.

---

##  Описание
- анализ и подготовка данных (очистка, обработка пропусков, устранение выбросов);
- обучение моделей регрессии и подбор гиперпараметров;
- оценка качества по метрикам регрессии (например, **MAE**, **R²**) и анализ ошибок.

Результатом является модель, способная прогнозировать цену ноутбука для новых конфигураций.


---

##  Предобработка
- удаление пустых/дублирующихся записей;
- удаление незначимых признаков;
