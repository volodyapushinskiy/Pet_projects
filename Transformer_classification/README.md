# Классификация тональности предложений (DistilBERT + Transformers)

В этом проекте решается задача **бинарной классификации тональности текста**.  
Текстовые данные представляют собой **список предложений из рецензий на фильмы**. Для каждого предложения требуется определить тональность:
- **1** — положительное настроение (*positive*)
- **0** — отрицательное настроение (*negative*)

В работе используется **предварительно обученная модель DistilBERT** из библиотеки `transformers`. Самый простой способ применить такую модель — использовать `pipeline`, который включает токенизацию, прогон через модель и получение предсказаний.

##  Описание
В ноутбуке показан подход, при котором:
1. Загружаем предварительно обученную модель **DistilBERT**.
2. Применяем её к предложениям с помощью `transformers.pipeline`.
3. Получаем предсказания по каждому предложению (positive/negative) и приводим их к меткам **1/0**.

---

##  Постановка задачи
**Вход:** одно предложение (строка)  
**Выход:** метка класса  
- `1` — положительная тональность  
- `0` — отрицательная тональность  

---

##  Используемая модель
- **DistilBERT** — компактная версия BERT, обученная на больших текстовых данных и хорошо подходящая для задач NLP.
- Используется библиотека **Hugging Face Transformers** и высокоуровневый интерфейс **pipeline** для анализа тональности.
