# ML / DL / NLP Pet Projects

Репозиторий с учебными и практическими проектами по машинному обучению, deep learning и NLP.
Более подробное описание проектов внутри самой папки

---

## Проекты

### 1) Оценка эмоциональной окраски отзыва (Sentiment Analysis, NLP)
**Задача:** определить тональность текста отзыва (например: negative / neutral / positive). 
**Папка** Orders_review

---

### 2) Классификация текста с помощью BERT (DL, NLP)
**Задача:** классификация на основе трансформеров.  
**Папка** Transformer_classification

---

### 3) Классификация email на spam / not spam (ML, NLP)
**Задача:** бинарная классификация писем.  
**Папка** Email_classification

---

### 4) Предсказание цены ноутбука (Regression, ML)
**Задача:** предсказать стоимость ноутбука по характеристикам.  
**Папка** Laptop_prices_prediction
